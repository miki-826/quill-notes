# Quill Notes

思考を、美しく残すためのミニマルなノートアプリケーション。

Quill Notes は、余計な装飾を削ぎ落とし、書くことそのものに集中するために設計された Web アプリケーションです。洗練されたモノトーンのデザインと、直感的な操作性を提供します。

## 特徴

- **洗練されたインターフェース**: 「非AI感」を追求した、清潔感のあるプロフェッショナルなデザイン。
- **高速な操作**: ページ遷移のないシングルページ設計で、思考を妨げません。
- **自動保存**: 入力と同時に LocalStorage へ自動保存されるため、保存忘れの心配がありません。
- **タグ管理**: カテゴリ分けを柔軟に行えるタグ機能を搭載。
- **全文検索**: 蓄積されたノートから、必要な情報を瞬時に見つけ出せます。
- **美しいタイポグラフィ**: 読みやすさにこだわったフォント選定（Inter / Noto Sans JP）。

## テクノロジースタック

- **言語**: HTML5, CSS3, JavaScript (Vanilla JS)
- **フォント**: Google Fonts (Inter, Noto Sans JP)
- **アイコン**: SVG
- **ストレージ**: ブラウザ LocalStorage

## 使い方

1. **ノート作成**: 左上の「+」ボタンをクリックして新しいノートを作成します。
2. **編集**: タイトルと本文を入力します。変更は自動的に保存されます。
3. **タグ付け**: エディタ上部の「タグを追加」からタグを設定できます（Enterで確定）。
4. **検索**: サイドバーの検索ボックスから、タイトルや本文の内容でフィルタリングできます。
5. **タグによる絞り込み**: サイドバーのタグをクリックすることで、特定のカテゴリのノートのみを表示できます。

## 開発コンセプト

「Quill（羽根ペン）」の名が示す通り、デジタルでありながらアナログのような手軽さと、道具としての美しさを両立させることを目指しました。モダンで信頼感のある UI/UX を追求しています。
